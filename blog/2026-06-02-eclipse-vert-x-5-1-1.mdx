We are extremely pleased to announce that Eclipse Vert.x version 5.1.1 has been released.

Since the release of Vert.x 5.1.0, quite a few bugs have been reported. We would like to thank you all for reporting these issues.

In addition, this release ships with Netty 4.2.15.Final that fixes a fair amount of vulnerabilities, details can be
found in the [Netty release page](https://netty.io/news/2026/06/01/4-2-15-Final.html).

The [5.1.1 release notes](https://github.com/vert-x3/wiki/wiki/5.0.1-Release-Notes) can be found on the wiki.

You can bootstrap a Vert.x 5 project using [start.vertx.io](https://start.vertx.io).

The release artifacts have been deployed to [Maven Central](https://search.maven.org/search?q=g:io.vertx%20AND%20v:5.0.1).

The [Vert.x eventbus JavaScript client library](https://www.npmjs.com/package/@vertx/eventbus-bridge-client.js) is now available
in a single location and it now usable standalone or it can easily be integrated with any frontend build tool.

That's it! Happy coding and see you soon on our user or dev [channels](https://vertx.io/channels).
