We are extremely pleased to announce that Eclipse Vert.x version 5.1.0 has been released.

Vert.x 5.1 comes with a set of new [exciting features](https://vertx.io/blog/whats-new-in-vert-x-5-1/), including
HTTP/3 and QUIC support.

As usual, our [examples](https://github.com/vert-x3/vertx-examples/) have been updated with 5.1.

The [5.1.0 release notes](https://github.com/vert-x3/wiki/wiki/5.1.0-Release-Notes) as well as
[deprecations and breaking changes](https://github.com/vert-x3/wiki/wiki/5.1.0-Deprecations-and-breaking-changes)
can be found on the wiki.

You can bootstrap a Vert.x 5 project using [start.vertx.io](https://start.vertx.io).

The release artifacts have been deployed to [Maven Central](https://search.maven.org/search?q=g:io.vertx%20AND%20v:5.1.0).

The [Vert.x 4 eventbus JavaScript client library](https://www.npmjs.com/package/@vertx/eventbus-bridge-client.js) is now available
in a single location and it now usable standalone or it can easily be integrated with any frontend build tool.

That's it! Happy coding and see you soon on our user or dev [channels](https://vertx.io/channels).
